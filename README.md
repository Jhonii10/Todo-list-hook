# Todo App

Todo App is a very basic to-do list app built with React and Bootstrap. With Todo App, you can add and delete, view your pending tasks, and mark tasks on your list as completed quickly and easily.

## Technologies

- ReactJS
- Javascript
- Css
- Bootstrap
